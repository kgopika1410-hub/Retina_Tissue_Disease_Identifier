## ⚡ Quick Start: Deploy in 5 Minutes

### Step 1: Verify Local Setup (2 min)
```bash
cd d:\Retina_Tissue
python setup_deployment.py
```
This checks:
- ✓ Models exist
- ✓ Dependencies installed
- ✓ Git repository initialized

### Step 2: Push to GitHub (2 min)
```bash
# One-time setup
git init
git add .
git commit -m "Initial commit: DR classifier for Render"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/retina-tissue.git
git push -u origin main
```

### Step 3: Deploy on Render (1 min)
1. Go to https://render.com → Sign up with GitHub
2. Click **New +** → **Web Service**
3. Select your `retina-tissue` repository
4. Click **Create Web Service**
5. Wait ~2-5 minutes
6. Get your live URL: `https://dr-classifier-xxxx.onrender.com`

## 📋 Important: Handling Model Files

Your trained models are too large for GitHub (~100-500MB each).

### Option A: Upload Models Manually (Easiest)
```bash

3. Upload model file via SFTP or download from cloud storage
```

### Option B: Use Cloud Storage (Recommended)
```bash
# Upload to AWS S3
```
